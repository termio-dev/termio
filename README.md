# Termio

**Modern terminal & SSH workspace manager for developers**

Termio is a desktop terminal app that helps you manage local shells and SSH sessions in powerful, organized workspaces — with built-in AI assistance and a clean, unified interface.

---

## ✨ Features

- **🖥 Local & SSH terminals**  
  Connect to local shells and remote servers in one place

- **🧩 Split-pane workspaces**  
  Run multiple sessions side-by-side with tabs and layouts

- **📁 Per-connection files**  
  Store scripts and notes directly inside each connection

- **⚡ Command composer**  
  Prepare and run multi-line commands safely

- **🤖 AI Copilot**  
  Get help with commands, debugging, and workflows directly in your terminal

- **🎨 Unified theme system**  
  Clean dark & light themes across terminal, UI, and editor

- **⭐ Favorites & organization**  
  Quickly access your most-used connections

- **⌨️ Command palette (Cmd+K)**  
  Navigate and execute actions instantly
